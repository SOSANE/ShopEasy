# Definition of Done (DoD)

Ci-dessous les critères pour qu'une user story ou une tâche est considérée comme terminée.

- Tous les critères d'acceptances reliés à la user story ou à la tâche sont respectées.
- Le code s'exécute sans erreurs.
- Tous les tests passent.
- La documentation relié à la user story ou la tâche a été mise à jour.
- Une revue de code a été accomplie par un ou plusieurs membres de l'équipe pour assurer la qualité et l'adhérence aux standard du code.
